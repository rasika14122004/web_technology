React is javaScript library developed by meta to build fast and interactive User interface.
It is a single page application(SPA) page does not reload only content change.
Example - Instagram , Facebook


Why do we use react ?
Answer ->
Problem with traditional js - fully page reload, complex dom manipulation, hard to manage large applications.
by using react - updates only required parts, use reusable components

we write everything html, css, js in only one file do not need to write different files

Core concept 
virtual DOM --> 
- React creates a copy of real DOM means virtual DOM
- When data changes react updates virtual DOM, updates only changed part in real DOM
- used for high performance

**Component**

1. Functional Component -

**How to use component **
    function App(){

    }



**Props**
- props means data passing from parent to child 
    //props -- parent
    function App(){
        return <student name = 'Siddhi'/>
    }

    //child 
    function student(props){
        return <h2>Name : {props.name} </h2>
    }
