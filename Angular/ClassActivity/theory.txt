---Data Binding ---- 13 April 2026
Data binding is a binding technique it connects UI to code
There are two types of data binding 
1. One way 
      1. interpolation --  syntax --{{value}}
      - Show data from ts to html 
      2. property binding --- Syntax-->  [property] = "value"
      - Bind html property to ts variable
      3. event binding --- Syntax --> (click) = 'function()'
      - call function on event 
   

2.two way -
 ngModel - sync data both ways ts to html and html to ts
      [(ngModel)] = 'Value'
